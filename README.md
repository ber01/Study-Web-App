# Study-Web-App
`Spring`과 `JPA`를 기반으로 웹 서비스 개발을 연습하는 저장소입니다.

## Dev
|도구|버전|
|:---:|:---:|
|Spring|Spring Boot 2.3.1.RELEASE|
|운영체제|Mac OS X|
|개발 툴|IntelliJ IDEA Ultimate|
|JDK|JDK 11|
|빌드 툴|Maven 3.6.3|
|데이터베이스| H2, PostgreSQL |

## Dependencies
- Spring Boot DevTools
- Lombok
- Spring Configuration Processor
- Spring Web
- Thymeleaf
- Spring Security
- Spring Data JPA
- H2 Database
- PostgreSQL Driver
- Java Mail Sender
